# Sugar SDK Demo App

To run this app you need to pass an Alchemy API key to the VITE_ALCHEMY_API_KEY environment variable.

#### Windows (Powershell)

```powershell
($env:VITE_ALCHEMY_API_KEY = "...") -and (npm run dev -w demo)
```

#### Linux, macOS (Bash)

```shell
VITE_ALCHEMY_API_KEY=... npm run dev -w demo
```
